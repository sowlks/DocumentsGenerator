# DocumentsGenerator
Fast generator open xml documents from template.

Create template by word processor. Use special tags for data management.
### Available tags:
- Value
- Rows
- GroupRows
- ExistIf

### Properties of tags:
- Condition
- Fit
- GroupCount
- Group
- Index
- OneString
- RemoveEmpty
- RowSpan
- Split
- Sum

See demonstration application for basic using.
